# Terrain

Vulkan terrain renderer. 
Implements CDLOD (Continuous Distance-Dependent Level of Detail) for smooth LOD transition:
	https://aggrobird.com/files/cdlod_latest.pdf
	https://svnte.se/cdlod-terrain

Control keys:
	w - forward
	s - backward
	1 - show/hide bounding boxes
	2 - toggle wireframe mode
	3 - show/hide water
